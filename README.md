# eslint-config-atomikui

The ESLint config for @atomikui/core

### Installation

```shell
# with npm
npm install --save-dev @atomikui/eslint-config-atomikui

# with yarn
yarn add --dev @atomikui/eslint-config-atomikui
```

### Add to your project's .eslintrc file

```json
{
  "parser": "babel-eslint",
  "extends": ["atomikui"]
}
```
